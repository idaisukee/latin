名詞第 1 曲用
===

第1曲用は、AとBとCに分類される。Aが多数であり、B, Cはごく少数である。

# A

|  | 主格 | 属格 |
| ------------- | ------------- | ------------- |
| 単数 | -a | -ae |
| 複数 | -ae | -arum |

# B

|  | 主格 | 属格 |
| ------------- | ------------- | ------------- |
| 単数 | -e | -ae |
| 複数 | -es | -arum |

# C

|  | 主格 | 属格 |
| ------------- | ------------- | ------------- |
| 単数 | -es | -ae |
| 複数 | -ae | -arum |
